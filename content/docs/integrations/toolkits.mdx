# Available Toolkits

Pantheon curates a set of Composio toolkits with descriptions and recommendations for common team workflows. Each toolkit bundles the most useful actions for a single service.

## Recommended toolkits

These four toolkits cover the most common integration use cases. They are highlighted in the **Recommended** tab when selecting toolkits.

### Google Sheets

| | |
|---|---|
| **Category** | Recommended |
| **Requires auth** | Yes (Google account) |
| **Actions** | Read Spreadsheet, Write Spreadsheet, Create Spreadsheet |

**What it does:** Read and write Google Sheets spreadsheets directly from Discord.

**Use cases:**

- Track project deliverables and metrics in a shared spreadsheet
- Log daily observations and activity data
- Pull budget numbers from a planning spreadsheet
- Build summary reports that update automatically

**Example prompts:**

```
Add today's metrics to the Q1 tracking spreadsheet
```

```
What does the budget tracker show for marketing?
```

```
Create a new spreadsheet called "Project Notes" with columns for date, topic, notes, and action items
```

<Callout variant="tip" title="Combine with custom skills">
Google Sheets works great alongside custom skills. Ask your agent to "export this week's data to my tracking spreadsheet" and it will read your skill data and write to Google Sheets in one step.
</Callout>

### Google Calendar

| | |
|---|---|
| **Category** | Recommended |
| **Requires auth** | Yes (Google account) |
| **Actions** | Create Event, List Events, Update Event |

**What it does:** Manage Google Calendar events -- create, read, and update scheduling entries.

**Use cases:**

- Schedule meetings and work sessions
- Block time for focused work or team activities
- Track deadlines and deliverable dates
- Set reminders for important milestones

**Example prompts:**

```
Schedule the design review for next Monday at 2 PM
```

```
What's on the team calendar this week?
```

```
Move the client call to Thursday -- the schedule is clearer
```

### Gmail

| | |
|---|---|
| **Category** | Recommended |
| **Requires auth** | Yes (Google account) |
| **Actions** | Send Email, List Emails, Read Email |

**What it does:** Send and read Gmail messages through your agents.

**Use cases:**

- Forward project summaries to stakeholders
- Email daily or weekly reports to team members
- Share updates with external collaborators
- Send notifications about important events

**Example prompts:**

```
Email today's project summary to john@example.com
```

```
Send the weekly status report to the team
```

```
Check if I have any emails from the client about the proposal
```

<Callout variant="warning" title="Email sends are real">
When your agent sends an email through Gmail, it sends from your actual Gmail account. Double-check recipients and content before confirming sends, especially for the first few uses.
</Callout>

## Toolkit selection tips

### Start small

Enable only the toolkits you will actively use. Each toolkit adds actions to your agent's tool list, and a smaller tool list means faster, more focused responses. You can always add more toolkits later.

### Recommended first

The four recommended toolkits (Google Sheets, Google Calendar, Gmail, Slack) cover 90% of integration use cases for most teams. Start with these and expand as needed.

### Combine with skills

The most powerful workflows combine Pantheon skills with integration actions:

| Skill | + Integration | Result |
|---|---|---|
| Custom data skill | Google Sheets | Auto-export data to spreadsheets |
| Scheduling skill | Google Calendar | Schedule tasks around availability |
| Reporting skill | Gmail | Email reports to stakeholders |
| Briefing skill | Slack | Cross-post updates to team channels |

# Tools

Pantheon tools are delivered through skills. Each skill gives your agent a focused capability and the workflow needed to execute it reliably.

## Tool inventory

Pantheon provides a set of core platform tools that every workspace can use:

| Tool | What it does | Best for |
|---|---|---|
| [Custom Skills](/docs/skills/custom-skills) | Create your own skills with AI-assisted generation | Tailoring your agents to your specific workflows |
| [Knowledge Files](/docs/tools/knowledge) | Upload documents your agents can reference | Providing context from existing docs, manuals, and reference material |
| [Memory Vault](/docs/tools/memory) | Persistent facts and context across conversations | Remembering preferences, decisions, and ongoing context |
| [Email](/docs/tools/email) | Send and receive email through your agents | Communicating with people outside Discord |
| [Secrets Vault](/docs/tools/secrets-vault) | Store API keys for authenticated external API calls | Connecting to external services securely |
| [Spending Cap](/docs/billing/spending-cap) | Set monthly usage limits | Controlling costs |

## How tools are managed

Tool access follows the same two-step model as skills:

1. **Global toggle** in **Settings > Skills** controls whether the tool is enabled for the workspace.
2. **Per-agent assignment** in **Settings > Agents** controls which agents can use each enabled tool.

An explicit inventory of tools plus clear policy on how access is controlled.

## Custom skills

Beyond the platform tools, you can create **custom skills** to extend your agents with domain-specific capabilities. Custom skills are written in Markdown (SKILL.md format) and can be generated with AI assistance. See [Custom Skills](/docs/skills/custom-skills) for details.

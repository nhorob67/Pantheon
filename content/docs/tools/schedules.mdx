# Schedules & Briefings

Schedules let you automate recurring messages from your agents. The main feature is the **daily briefing** -- a configurable summary posted to your Discord channel on a schedule you define.

## Daily briefing

The briefing is a single scheduled message that combines multiple sections. Configure it from **Settings > Schedules**.

### Briefing sections

Toggle each section on or off. The available sections depend on the custom skills you have enabled. Common examples include:

| Section | What it includes |
|---|---|
| **Summary** | A digest of recent activity, key updates, and pending items |
| **Custom reports** | Data pulled from your configured skills and knowledge files |

### Configuration

| Setting | Options |
|---|---|
| **Enable/disable** | Toggle the entire briefing on or off |
| **Send time** | Time of day (e.g., 06:30) |
| **Timezone** | America/Chicago, America/Denver, America/New\_York, America/Los\_Angeles |
| **Discord channel** | The channel ID where the briefing is posted |

<Callout variant="info" title="Finding your channel ID">
To get a Discord channel ID: enable Developer Mode in Discord (User Settings > Advanced), then right-click the channel and click "Copy Channel ID".
</Callout>

## Per-agent cron jobs

In addition to the briefing system, individual agents can have their own scheduled jobs. You can configure any agent to run a scheduled task at a specific time and frequency. Common examples:

- Morning summary briefings
- Daily status reports
- Weekly digest messages
- Recurring data collection tasks

Configure per-agent cron jobs from **Settings > Agents** on each agent's card. Times are in your workspace's configured timezone.

## Cost considerations

Each scheduled run consumes API tokens:

- A typical briefing: ~1,500--3,000 tokens
- Data-heavy reports: ~2,000--4,000 tokens
- Full briefing (all sections): ~3,000--6,000 tokens

Factor this into your [spending cap](/docs/billing/spending-cap) and usage estimates. Disabling unused schedules is the easiest way to reduce costs.

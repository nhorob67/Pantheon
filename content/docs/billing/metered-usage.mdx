## How metering works

Every interaction with your AI agent consumes tokens -- the unit of measurement for AI model usage. Pantheon tracks token usage daily and bills it as a separate line item on top of your $50/month subscription.

### What are tokens?

Tokens are the building blocks of AI text processing. Roughly:

- **1 token** ~ 4 characters of English text
- **100 tokens** ~ 75 words
- A typical question and answer uses **500--2,000 tokens**

Both the message you send and the response your agent generates count toward token usage.

## Usage by task

Different tasks consume different amounts of tokens:

| Task | Typical Token Usage |
|------|-------------------|
| Simple question/answer | 500--1,000 |
| Data lookup or retrieval | 1,500--2,500 |
| Multi-source comparison | 2,000--4,000 |
| Report generation | 1,000--2,000 |
| Scheduled briefing | 1,500--3,000 |
| Document analysis | 1,000--2,000 |
| Image analysis (OCR) | 3,000--5,000 |
| Complex multi-step task | 2,000--4,000 |

## Tracking your usage

### Dashboard

Your Pantheon dashboard shows real-time usage analytics:

- **Daily token consumption** -- Bar chart showing tokens used per day
- **Cost breakdown** -- Running total for the current billing period
- **Per-model breakdown** -- Token usage split by AI model
- **Trend lines** -- See how your usage changes over time

### Usage table

The `api_usage` table tracks daily metering per model:

- Date
- Model used
- Input tokens consumed
- Output tokens generated
- Estimated cost

Access your usage data anytime from **Settings > Usage** or the main dashboard.

## Reducing usage

If you want to keep costs down:

- **Disable unused cron jobs** -- Automated briefings run on schedule even if you're not reading them
- **Use fewer agents** -- Each agent in a multi-agent setup consumes tokens independently
- **Be specific** -- Clear, direct questions use fewer tokens than vague ones
- **Use text input** instead of images when possible -- image processing uses significantly more tokens

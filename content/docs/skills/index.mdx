# Skills

Skills define what your AI agents can do. Each skill is a Markdown-based instruction set (SKILL.md format) that teaches an agent how to perform a specific task — from looking up FAQs to generating reports to processing inbound requests.

## All skills are custom

Pantheon's skill system is 100% custom. You create skills tailored to your exact use case. There are no locked-in built-in skills.

### Three ways to create skills

| Method | How it works |
|--------|-------------|
| **Skill Forge** | Write a skill from scratch using the built-in Markdown editor |
| **AI Generation** | Describe what you want in plain English and Pantheon generates the skill |
| **Extension Marketplace** | Install pre-built skills shared by the community |

<Callout variant="info" title="Included on every plan">
Custom skills are included in the $50/month subscription. The only variable cost is metered API usage when agents use skills to process messages.
</Callout>

## How skills work

Each skill contains:

- **Instructions** — Detailed guidance that tells the agent how to perform the task, what data to look for, and how to format output
- **Tools** — Access to APIs, databases, or file creation capabilities needed to complete the task
- **Configuration** — Any settings or parameters the skill needs

When you ask an agent a question, it determines which of its assigned skills applies and follows the skill's instructions to produce a response.

## Managing skills

### Creating a skill

1. Go to **Settings > Skills**
2. Click **Create Skill** or **Generate with AI**
3. Write or generate the skill content
4. Save

### Assigning skills to agents

Skills are assigned per-agent. Each agent only has access to its assigned skills.

1. Go to **Agents** in the sidebar
2. Select the agent you want to configure
3. Under **Skills**, toggle each skill on or off
4. Save

This lets you create focused specialists:

- A **Support Lead** with "FAQ Lookup" and "Ticket Triage" skills
- A **Research Analyst** with "Web Search" and "Report Generator" skills
- A **General Assistant** with all skills for catch-all questions

### Version history

Every skill edit creates a new version. You can view the version history and roll back to any previous version from the skill editor.

## Skills and agents

If you run a [multi-agent setup](/docs/agents/skills-per-agent), each agent gets its own subset of skills:

| Configuration level | Where to change it | What it controls |
|---|---|---|
| Skill creation | Settings > Skills | Creates and manages the skill definition |
| Per-agent assignment | Agents > [Agent] > Skills | Which agents can use the skill |

A skill created in Settings > Skills is available for assignment to any agent. Deleting a skill removes it from all agents.

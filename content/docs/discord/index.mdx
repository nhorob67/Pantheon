## Why Discord?

Pantheon delivers your AI agents through Discord because it's the best fit for how teams actually work:

- **Mobile-first** -- Discord runs on your phone, tablet, and desktop. Ask your agents questions from anywhere.
- **Already familiar** -- Many teams already use Discord for communication. No new app to learn.
- **Multi-channel** -- Organize conversations by topic. Put different workflows in different channels.
- **Threaded conversations** -- Keep discussions organized without losing context. Ask a follow-up question in a thread without cluttering the main channel.
- **File sharing built in** -- Your agents can send spreadsheets, charts, and PDFs directly in the conversation. Download files right to your phone.
- **Notifications you control** -- Set up push notifications for the channels that matter. Get alerted for important updates but mute channels on weekends.

## One server, one workspace

Each Pantheon workspace connects to **one Discord server**. When you sign up and complete onboarding, the Pantheon bot joins your server and is ready to respond to messages.

If you don't already have a Discord server, you'll create one during setup. If you have an existing server for your team, you can add the Pantheon bot to it.

## What you can configure

Pantheon gives you control over how your agents operate within Discord:

### Bot setup

Add the Pantheon bot to your Discord server during onboarding with a single click. No bot tokens or Developer Portal steps needed. See the [Server Setup](/docs/discord/server-setup) guide for step-by-step instructions.

### Channel configuration

Map specific AI agents to specific Discord channels. For example, bind your Research agent to `#research` and your Support agent to `#support`. Any channel without an explicit binding is handled by your default agent. See [Channel Configuration](/docs/discord/channel-config) for details.

### Agent bindings

If you're running a multi-agent setup, each agent can be assigned to one or more channels. This keeps conversations focused -- each agent only responds in its assigned channels, and your default agent covers everything else plus DMs.

Learn more about agents in the [AI Agents](/docs/agents) section.

## How it works in practice

Once your workspace is set up, using Pantheon is as simple as typing a message in Discord:

```
What's the status of our project?
```

```
Summarize the latest report
```

```
Draft a response to this client request
```

Your agent responds in the same channel, with formatted tables, data citations, and timestamps. It can also generate and attach files -- spreadsheets, charts, PDFs -- directly in the conversation.

<Callout variant="tip" title="Works from any device">
Anything you can do from the Discord desktop app, you can do from the mobile app. Pantheon responds the same way regardless of how you connect.
</Callout>

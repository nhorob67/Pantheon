## Why Discord?

FarmClaw delivers your AI assistant through Discord because it's the best fit for how farmers actually work:

- **Mobile-first** — Discord runs on your phone, tablet, and desktop. Ask for grain bids from the combine cab or check spray windows from the pickup.
- **Already familiar** — Many farmers and farm families already use Discord. No new app to learn.
- **Multi-channel** — Organize conversations by topic. Put grain talk in one channel, weather in another, scale tickets in a third.
- **Threaded conversations** — Keep discussions organized without losing context. Ask a follow-up question in a thread without cluttering the main channel.
- **File sharing built in** — Your assistant can send spreadsheets, charts, and PDFs directly in the conversation. Download a bid comparison Excel file right to your phone.
- **Notifications you control** — Set up push notifications for the channels that matter. Get alerted for morning weather but mute the grain desk on weekends.

## One server, one workspace

Each FarmClaw workspace connects to **one Discord server**. When you sign up and complete onboarding, the FarmClaw bot joins your server and is ready to respond to messages.

If you don't already have a Discord server, you'll create one during setup. If you have an existing server for your operation, you can add the FarmClaw bot to it.

## What you can configure

FarmClaw gives you control over how your assistant operates within Discord:

### Bot setup

Add the FarmClaw bot to your Discord server during onboarding with a single click. No bot tokens or Developer Portal steps needed. See the [Server Setup](/docs/discord/server-setup) guide for step-by-step instructions.

### Channel configuration

Map specific AI agents to specific Discord channels. For example, bind your Grain Specialist to `#grain-desk` and your Weather agent to `#weather`. Any channel without an explicit binding is handled by your default agent. See [Channel Configuration](/docs/discord/channel-config) for details.

### Agent bindings

If you're running a multi-agent setup, each agent can be assigned to one or more channels. This keeps conversations focused — your Grain Specialist only responds in the grain channel, your Scale Ticket Clerk handles the tickets channel, and your general Farm Assistant covers everything else plus DMs.

Learn more about agents in the [AI Agents](/docs/agents) section.

## How it works in practice

Once your workspace is set up, using FarmClaw is as simple as typing a message in Discord:

```
What's corn at today?
```

```
Spray window for tomorrow morning?
```

```
Log a scale ticket — 47k of beans, 13.2 moisture, going to CHS
```

Your assistant responds in the same channel, with formatted tables, data citations, and timestamps. It can also generate and attach files — spreadsheets, charts, PDFs — directly in the conversation.

<Callout variant="tip" title="Works from any device">
Anything you can do from the Discord desktop app, you can do from the mobile app. FarmClaw responds the same way regardless of how you connect.
</Callout>
